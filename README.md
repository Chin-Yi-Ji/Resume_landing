## 🚀 快速開始

### 安裝依賴

```bash
npm install
```

### 啟動開發伺服器

```bash
npm start
```

網站將在 [http://localhost:3000](http://localhost:3000) 開啟。

### 建置生產版本

```bash
npm run build
```
